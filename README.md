# Calculadora de Conversión de Bases (2, 8, 10, 16)

Este proyecto es un sistema desarrollado en **Java** que permite realizar conversiones numéricas de forma rápida y sencilla entre diferentes sistemas de numeración.

### 🔄 Sistemas de Numeración Soportados:
* **Binario** (Base 2)
* **Octal** (Base 8)
* **Decimal** (Base 10)
* **Hexadecimal** (Base 16)

---

## 🛠️ Tecnologías Utilizadas

* **Lenguaje:** Java
* **Interfaz Gráfica:** JavaFX
* **IDE de Desarrollo:** NetBeans

---

## 🚀 Requisitos de Ejecución

Para poder ejecutar e interactuar con este programa en tu entorno local, asegúrate de contar con los siguientes elementos:

1. **Java Development Kit (JDK):** Versión compatible con tu IDE.
2. **JavaFX:** Es necesario tener instalado y configurado el SDK de JavaFX en tu equipo.
3. **IDE Compatible:** NetBeans (recomendado) o cualquier otro IDE compatible con proyectos Java y JavaFX (como IntelliJ IDEA o Eclipse).
