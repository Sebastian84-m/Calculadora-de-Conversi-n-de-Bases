#CAlCULADORA DE CONVERSION DE BASES (2,8,10,16)

Este proyecto es un sistema desarrollado en JAVA, el cual permite convertir numeros entre distintos sistemas de numeracion como lo son: 
-Binal
-Octal
-Decimal
-Hexadecimal

El sistema fue desarrollado en NetBeans, para ejecutar el programa es necesario tener JavaFX instalado en el computador junto con un IDE compatible con JAVA
